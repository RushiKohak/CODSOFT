# Customer Churn Prediction

## Project Overview
This project predicts whether a customer will leave a bank service or not using machine learning classification algorithms.

## Objective
To build a model that identifies customers who are likely to churn based on their account and personal details.

## Algorithms Used
- Logistic Regression
- Random Forest

## Dataset
Dataset used: Churn_Modelling.csv  
The dataset contains customer information such as credit score, geography, gender, age, balance, and salary.
